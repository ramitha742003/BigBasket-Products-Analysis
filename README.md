# BigBasket Products Analysis

Designed and trained the supervised learning model and used visualization tools like matplotlib or seaborn to visualize the data.

Implemented a Streamlit web application to display the dashboard and get specific product's information from the dataset.

Created a interactive dashboard using Power BI.

Tools used: Jupyter notebook, Python, Streamlit, Power BI.

# Kaggle Dataset

https://www.kaggle.com/datasets/amrit0611/big-basket-product-analysis
